# 노안탈출
 Google Vision, AWS Polly 를 활용한 Image to Speech 서비스로, 노안 때문에 잘 안 보이는 글자를 카메라로 찍으면 사진에 포함된 글자를 읽어 주는 서비스 개발

## 사용 기술
- Spring Boot
> AWS-SDK, GCP-SDK 활용<br>

- Android
> HTTP 통신<br>
> MP3 Streaming
